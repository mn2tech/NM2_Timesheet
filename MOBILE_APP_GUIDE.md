# Mobile App Installation Guide

Your NM2Timesheet application is now a **Progressive Web App (PWA)** that can be installed on mobile devices!

## ðŸ“± How to Install on Mobile

### For Android (Chrome/Edge)

1. **Open the app** in Chrome or Edge browser on your Android device
2. **Look for the install prompt** at the bottom of the screen
   - If you see "Add to Home screen" or "Install app", tap it
3. **Or manually install:**
   - Tap the **menu** (three dots) in the browser
   - Select **"Add to Home screen"** or **"Install app"**
   - Tap **"Add"** or **"Install"** to confirm
4. The app icon will appear on your home screen
5. Tap the icon to open the app in full-screen mode

### For iPhone/iPad (Safari)

1. **Open the app** in Safari browser on your iOS device
2. **Tap the Share button** (square with arrow pointing up)
3. **Scroll down** and tap **"Add to Home Screen"**
4. **Edit the name** if desired (default: "NM2Timesheet")
5. **Tap "Add"** in the top right
6. The app icon will appear on your home screen
7. Tap the icon to open the app in full-screen mode

### For Desktop (Chrome/Edge)

1. **Open the app** in Chrome or Edge browser
2. **Look for the install icon** in the address bar (usually a "+" or download icon)
3. **Click "Install"** when prompted
4. The app will open in its own window, like a native app

## ðŸ”§ Troubleshooting

### Install option not showing?

**Android:**
- Make sure you're using Chrome or Edge
- The site must be served over HTTPS (or localhost)
- Try clearing browser cache and reloading

**iPhone:**
- Make sure you're using Safari (not Chrome)
- The site must be served over HTTPS
- Try closing and reopening Safari

### App not working offline?

- The app needs to be installed first
- Some features require internet connection (login, saving, etc.)
- Basic viewing works offline after initial load

### Icon not appearing?

- Wait a few seconds after installation
- Check your home screen folders
- Try reinstalling the app
